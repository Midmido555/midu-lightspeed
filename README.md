# Midu Lightspeed

A futuristic AI-powered platform for creative minds.

## Features

- 🎬 Cinematic Scripts Generator
- 🎤 Voice Narration
- 🎥 AI Video Generator
- 📘 Smart PDF Creator
- 💡 Viral Idea Engine
- 🌍 Multi-language support (English & Arabic)

## Tech Stack

- Next.js 14
- Tailwind CSS
- Framer Motion
- ShadCN UI
- OpenAI API
- Vercel Deployment

## Getting Started

```bash
git clone https://github.com/yourusername/midu_lightspeed.git
cd midu_lightspeed
npm install
npm run dev
```
